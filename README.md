# 🌦️ Premium Glass Weather App

A modern and professional Weather Application built using **React JS** with a premium glassmorphism UI design.
This app allows users to search for any city and view real-time weather details with a smooth animated interface.

## 🚀 Live Features

✨ Search weather by city name
✨ Real-time temperature display
✨ Animated gradient temperature text
✨ Glassmorphism UI design
✨ Auto zoom-in / zoom-out animation
✨ Hover glow effects
✨ Responsive design (Mobile Friendly)
✨ Animated background particles

## 🛠️ Built With

* ⚛️ React JS
* 🎨 CSS3 (Glassmorphism + Animations)
* 🌍 OpenWeather API (for live weather data)

## ⚙️ Installation & Setup

1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/weather-report-app.git
```

2️⃣ Navigate to project folder

```bash
cd weather-report-app
```

3️⃣ Install dependencies

```bash
npm install
```

4️⃣ Start the development server

```bash
npm start
```

App will run on:

```
http://localhost:3000
```
